# 💬 Real-Time Chat Backend

A lightweight real-time chat backend built using **FastAPI**, **WebSockets**, **PostgreSQL**, and **Docker**.

---

## 🚀 Features
- ⚡ Real-time messaging with WebSockets  
- 🔐 Secure user authentication  
- 🗄 PostgreSQL + SQLAlchemy ORM  
- 🧩 Clean modular folder structure  
- 🐳 Fully Dockerized  
- 📘 Auto API docs available at `/docs`

---

## 🛠 Tech Stack
**FastAPI • WebSockets • Python • PostgreSQL • SQLAlchemy • Docker • Uvicorn**
